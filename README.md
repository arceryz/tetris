# Tetris

<img align="left" style="width:350px" src="https://github.com/arceryz/tetris/blob/master/demo.gif" width="380px">

This is a tetris proejct made by Nils en Timothy voor Game Programming 1.
You can play with 1 player or 2 players. There are extra SFX and screen shake
features to make it more satisfying.

---
<br><br>

# Features

- Tetris with 1 player mode classic style
- 2 player mode for extra strategy
- CRT effect shader for classic retro era
- Screen shake effect
- Sound effects
- Settings menu
- Fancy title menu
- Rainbow colors!

In 2-player mode there is a secret technique where you can merge two tetris blocks if you time it correctly.
